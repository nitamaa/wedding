#  微婚礼请柬 小程序

由于审核越来越严不想维护了

-  本小程序基于微信云开发-不需要服务器成本
-  支持自定义页面内容，图片、地址、视频等
-  没有内容审核机制
-  可发送单人观看 也可生成图片分享给所有人看
-  每个页面文字都是可以自定义进场效果 大小等
